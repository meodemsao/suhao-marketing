# 📁 Assets — Su Hào Interior

Thư mục chứa toàn bộ tài nguyên marketing & brand.

## Cấu trúc

```
assets/
├── brand/                    # Brand Identity
│   ├── logo/                 # Logo files (SVG, PNG, ICO)
│   ├── colors/               # Color swatches, palette files
│   ├── typography/           # Font files (nếu cần offline)
│   └── guidelines/           # Brand Guidelines PDF
│
├── social-media/             # Social Media Kit
│   ├── facebook/             # Avatar, cover, post templates
│   ├── tiktok/               # Avatar, banner
│   ├── youtube/              # Avatar, banner, thumbnails
│   └── zalo/                 # Zalo OA assets
│
├── content/                  # Content gốc
│   ├── photos/               # Ảnh chụp
│   │   ├── projects/         # Ảnh công trình (theo tên dự án)
│   │   ├── workshop/         # Ảnh xưởng sản xuất
│   │   ├── team/             # Ảnh đội ngũ
│   │   └── materials/        # Ảnh vật liệu
│   ├── videos/               # Video gốc
│   │   ├── tours/            # Tour công trình
│   │   ├── asmr/             # ASMR xưởng gỗ
│   │   ├── testimonials/     # Feedback khách hàng
│   │   ├── timelapse/        # Timelapse thi công
│   │   └── behind-the-scenes/ # Hậu trường
│   └── graphics/             # Đồ họa đã thiết kế
│       ├── thumbnails/       # Thumbnail video
│       ├── infographics/     # Infographic tips
│       ├── carousels/        # Carousel posts
│       └── banners/          # Banner quảng cáo
│
├── templates/                # Template tái sử dụng
│   ├── canva/                # File Canva (.canva links)
│   ├── social-posts/         # Template bài đăng
│   ├── blog-covers/          # Ảnh bìa blog
│   └── presentations/        # Slide trình bày
│
└── marketing-collateral/     # Tài liệu in ấn
    ├── business-cards/       # Danh thiếp
    ├── letterhead/           # Giấy tiêu đề
    ├── email-signatures/     # Chữ ký email
    └── brochures/            # Tờ rơi / catalogue
```

> **Lưu ý:** Logo sẽ được cung cấp sau. Khi có logo, đặt vào `brand/logo/` với các format:
> - `logo-primary.svg` — Logo chính (vector)
> - `logo-dark.svg` — Logo trên nền tối
> - `logo-light.svg` — Logo trên nền sáng
> - `logo-icon.svg` — Icon only (không có text)
> - `logo-primary.png` — PNG 1024x1024
> - `favicon.ico` — Favicon 32x32
