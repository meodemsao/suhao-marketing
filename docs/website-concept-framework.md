## BRAND POSITIONING (Nền tảng thiết kế)

| Yếu tố | Định hướng |
|---------|-----------|
| **Tone** | Luxury + Minh bạch — sang trọng nhưng không xa cách |
| **Phong cách** | Exaggerated Minimalism — đen/trắng/vàng gold, typography lớn, negative space |
| **Typography** | Cinzel (heading luxury) + Josefin Sans (body hiện đại) |
| **Màu sắc** | Primary `#171717`, CTA `#D4AF37` (vàng gold), BG `#FFFFFF`, Muted `#404040` |
| **Ảnh** | 100% ảnh thực, zero stock — thể hiện CRAFTSMANSHIP |
| **Ngôn ngữ** | Không "giảm giá", không "sale" — dùng "đầu tư", "giá trị" |

## THÀNH PHẦN XUYÊN SUỐT (Global Components)

| Component | Mô tả |
|-----------|-------|
| **Navbar** | Floating, glass effect, logo trái, menu giữa, CTA "Đặt lịch tư vấn" phải. Scroll → nền đặc hơn |
| **Footer** | Logo + info liên hệ + social links + newsletter signup + sitemap links |
| **Zalo Chat Widget** | Float bottom-right, pulse animation khi load |
| **Back to Top** | Hiện khi scroll > 500px |
| **Loading** | Skeleton screens cho images + content |
| **Scroll Animations** | Fade-in-up khi element vào viewport. Respect `prefers-reduced-motion` |

## HIỆU ỨNG & ANIMATION

| Hiệu ứng | Áp dụng | Kỹ thuật |
|-----------|---------|----------|
| Scroll reveal | Mọi section | IntersectionObserver, translateY(30px) → 0, opacity 0 → 1, 400ms |
| Parallax nhẹ | Hero background | transform: translateY(scrollY * 0.3) |
| Counter animation | Số liệu "Về chúng tôi" | Count-up từ 0, duration 2s, trigger khi visible |
| Hover card | Portfolio grid | Scale 1.02, shadow tăng, overlay fade-in, 200ms |
| Before/After slider | Chi tiết công trình | Drag handle, clip-path |
| Smooth scroll | Navigation links | scroll-behavior: smooth |
| Navbar transition | Khi scroll | Backdrop-blur tăng, background-color transition |
| Image lazy load | Tất cả ảnh | loading="lazy" + skeleton placeholder |

> [!IMPORTANT]
> Mọi animation PHẢI respect `prefers-reduced-motion: reduce` — tắt hoặc giảm animation cho người nhạy cảm.
